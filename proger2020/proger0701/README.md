计算过程：每月 300 公积金为例

- 一年后，公积金账户余额为

  300 * 12 *2 = 7200 （公司和个人对半缴纳）

- 公积金月平均余额 为：

  (600 + 1200 + 1800 +...+ 7200)/12 = 46800/12 = 3900

- 公积金贷款额度为：

  3900\*15 = 58500

* 两年后，公积金账户余额为

  300 * 12 *2 \*2 = 14400 （公司和个人对半缴纳）

* 公积金月平均余额 为：

  (7800 +...+ 14400)/12 = 133200/12 = 11100

* 公积金贷款额度为：

  10800\*15 = 162000

- 三年后，公积金账户余额为

  300 * 12 *2 \*3 = 21600 （公司和个人对半缴纳）

- 公积金月平均余额 为：

  (15000 +...+ 21600)/12 = 219600/12 = 18300

- 公积金贷款额度为：

  18000\*15 = 27000
